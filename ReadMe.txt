To run this program download test.csv and train.csv from zip file along with source code.
Put them in one directory.
(if you are using csv files from bb9, then delete the first row)
Open source code with some python IDE that is ready for machine learning projects.
set k and n values to whatever you like and wait for it to compile and output accuracy rate of prediction to you.

P.S.
It might take some time depending on values that you use.
